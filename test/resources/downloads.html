eAIP ISO Image

The latest loadable eAIP ISO Image file is on the top of the list. Please note the first date of validity. Check also the publication and AIRAC effective dates at AIC A web site (https://ais.fi/ais/aica/AipAicA_en.htm)

The file will not be renewed on such AIRAC effective dates when a new AIP AMDT is not published.

VFR reporting points

Finavia has used GPX format to describe VFR reporting points published on WAYPOINTS AND FIXES product for each aerodrome in AIP, AD 2.15. This GPX file contains all VFR waypoints published in the AIP and is updated whenever there are changes to the AIP.
